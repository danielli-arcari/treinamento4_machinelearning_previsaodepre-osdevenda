## 📌 Sobre o Projeto

Este projeto aplica técnicas de **Machine Learning** para prever o preço de venda de automóveis com base em suas características técnicas e comerciais. A análise cobre desde a exploração e limpeza dos dados até o treinamento e avaliação de modelos preditivos, demonstrando um pipeline completo de Ciência de Dados.

> 💡 *"Transformar dados brutos de automóveis em inteligência de precificação — do zero ao modelo."*

---

## 🎯 Objetivos

- Explorar e entender o dataset de características de veículos
- Realizar limpeza e pré-processamento dos dados
- Aplicar análise exploratória (EDA) para identificar padrões e correlações
- Treinar modelos de regressão para prever preços de venda
- Avaliar e comparar a performance dos modelos

## 📊 Pipeline do Projeto

```
📁 Dataset de Carros
       │
       ▼
🔍 Análise Exploratória (EDA)
   ├── Distribuição de preços
   ├── Correlação entre variáveis
   └── Identificação de outliers
       │
       ▼
🧹 Pré-processamento
   ├── Tratamento de valores nulos
   ├── Encoding de variáveis categóricas
   └── Normalização / Escalonamento
       │
       ▼
🤖 Modelagem ML
   ├── Treinamento do modelo de regressão
   ├── Validação cruzada
   └── Avaliação de métricas (R², MAE, RMSE)
       │
       ▼
📈 Resultados & Insights
```
